## ss-bash-centos-firewalld
=======

#### 服务器端口流量管理脚本

* 目前只支持统计ipv4流量
* 经过turingttc的修改，可以在centos服务器上使用流量统计功能
* 经过Bricdaii的修改，改写为支持CentOS7中默认Firewall-Cmd
